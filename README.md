# PNP-P3P
## Problem Statement
Estimate the pose of a calibrated camera when knowing N or 3 points.

## User's guide
1. The main algorithms are implemented in `pnp.py` and `p3p.py`
2. `python run_p3p` to run p3p algorithm.
3. `python run_pnp` to run pnp algorithm.
